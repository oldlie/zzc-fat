# zzc基金分析工具

#### 介绍
zzc基金分析工具

#### 软件架构
1.  UI: vue3
2.  Backend: Electron
3.  前后端采用事件的方式通信


#### 安装教程

1.  去到fund-ui执行```npm install```安装依赖
2.  执行```npm run build```将前端项目编译到app的指定文件夹
3.  去到app文件夹执行```npm install```安装依赖
4.  执行```npm start```启动程序
